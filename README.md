# Итеративные методы решения системы линейных уравнений

В корневой директории команда `make` создаст исполнимые файлы: `iterative_slover` и `tests`.

## Основная программа `iterative_slover`

Использование:
```bash
./iterative_slover <jacobi|seidel|relaxation>
```
Аргумент должен принимать одно из трех значений: `jacobi`, `seidel` или `relaxation`
для решения методом Якоби, методом Зейделя и методом релаксации соответственно.

Программа читает данные из файла `data.dat` и записывает решение в `result.dat`. На экран выводится модуль вектора невязки.

## Программа `tests`

Генерирует случайную систему порядка n=10000  и решает ее тремя методами. Выводит вектор невязки для каждого метода.
